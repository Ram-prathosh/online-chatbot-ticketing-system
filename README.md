# 🤖 Online Chatbot-Based Ticketing System

An **Online Chatbot-Based Ticketing System** is a web application that allows users to create and manage support tickets through a chatbot interface. The chatbot helps users quickly submit issues, track ticket status, and receive responses from administrators.

This project is built using **Python (Flask) for backend** and **HTML, CSS, JavaScript for frontend**.

## 📌 Features

* 💬 Chatbot interface for user interaction
* 🎫 Create support tickets easily
* 📋 View and manage submitted tickets
* 👨‍💻 Admin panel to monitor issues
* ⚡ Fast and simple web interface
* 🌐 Runs locally on browser

## 🛠 Technologies Used

* **Python**
* **Flask**
* **HTML**
* **CSS**
* **JavaScript**
* **Bootstrap (optional)**

## ⚙️ Installation

Follow these steps to run the project on your system.

### 1️⃣ Clone the repository

```
git clone https://github.com/yourusername/online-chatbot-ticketing-system.git
```

### 2️⃣ Navigate to the project folder

```
cd online-chatbot-ticketing-system
```

### 3️⃣ Install dependencies

```
pip install flask
```

or

```
pip install -r requirements.txt
```

---

## ▶️ Running the Application

Run the following command:

```
python app.py
```

After running, open your browser and go to:

```
http://127.0.0.1:5000
```

## 💡 How It Works

1. User opens the chatbot interface.
2. User describes the issue.
3. The chatbot collects information.
4. A ticket is created and stored.
5. Admin can review and resolve the ticket.
